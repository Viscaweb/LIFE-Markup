How to divide some sections into columns ?

You can create columns to split a section into multiple parts.

Add a div with class .row with other divs for columns inside, with class .col-sm-x:

Exemple for columns of 50% :

By default, there are 12 columns in a page, so for 50%, each columns will take 6 (.col-sm-6)

                        
                            <div class="row">
                                <div class="col-sm-6">...</div>
                                <div class="col-sm-6">...</div>
                            </div>
